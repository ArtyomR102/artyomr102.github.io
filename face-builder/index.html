Unicode emoji generator ALPHA

This is an alpha-version! Quite usable, but buggy and ugly

  1. Click on a symbol or empty space to select it
  2. Round buttons select different base symbols
  3. Checkboxes select combining symbols above and below the base symbol
  4. Mirror mode makes symbols appear mirrored (thx CO)
  5. Strict mode makes font-unsupported symbols disappear (doesn't work yet tho)

NOTE Font family and size affect preview only! Should be obvious, but needs to be said for a certain kind of people. And you guessed, doesn't work yet (0∇0)
